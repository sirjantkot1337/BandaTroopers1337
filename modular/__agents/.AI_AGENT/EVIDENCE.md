# EVIDENCE

## E-001: Активная задача не зафиксирована
- Этот файл должен содержать только факты и ссылки на raw outputs для текущей задачи.
- Если активной задачи нет, файл остается в baseline-состоянии.

## E-002: Правило хранения evidence
- Сырые выводы команд, тестов и проверок класть в `logs/`.
- В этом файле оставлять только краткие выводы и ссылки на сырые артефакты.
