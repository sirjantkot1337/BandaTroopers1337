RTO Support: памятка игроку
modular/rto_support/__docs/RTO_SUPPORT_PLAYER_GUIDE.md

1. Что у тебя есть

RTO получает:

  • RTO binoculars
  • RTO sling pouch с жёсткой связкой к этим биноклям

Кнопки RTO видны только пока бинокль у тебя в руке.

2. Базовый цикл для боевых пакетов

Для Mortar, CAS, Heavy Strike играй так:

  1. достань RTO-бинокль в руку;
  2. нажми Выбрать пакет поддержки;
  3. выбери пакет;
  4. нажми кнопку сектора наведения;
  5. во время зума через бинокль поставь сектор Ctrl+Click;
  6. нажми нужную поддержку;
  7. снова через бинокль наведи точку Ctrl+Click.

Коротко:

взял бинокль -> выбрал пакет -> развернул сектор -> выбрал абилку -> навёл через бинокль

Важно:

  • Mortar теперь работает через длинное окно сектора примерно на 100 секунд, после чего уходит в тяжёлую recovery-фазу примерно на 300 секунд;
  • CAS теперь держит темп ожидания примерно 10-20 секунд между ударами;
  • Heavy Strike тоже сведён к ожиданию примерно 10-20 секунд между ударами, но всё ещё остаётся самым тяжёлым боевым пакетом по recovery зоны;
  • у Mortar интервалы между выстрелами ещё меньше.

3. Logistics

Logistics работает без сектора.

Порядок:

  1. возьми бинокль в руку;
  2. выбери Logistics;
  3. нажми нужный сброс;
  4. наведи точку через бинокль Ctrl+Click.

У Logistics сейчас есть:

  • Supply drop
  • Mine crate drop
  • Mini-sentry drop
  • Full sentry drop

Важно:

  • Logistics — самый долгий пакет по кулдаунам;
  • это отложенная поддержка для снабжения и удержания позиции, а не для быстрого спама.

4. Utility-режимы

Координаты

  • это не абилка с cooldown;
  • это постоянный режим;
  • Ctrl+Click просто пишет координаты точки;
  • лазерной точки у этого режима нет.

Лазерная отметка

  • это тоже постоянный режим без cooldown;
  • Ctrl+Click ставит или переносит живую подсветку;
  • лазер держится только пока ты реально продолжаешь вести цель через бинокль.

5. Как читать кнопки

Кнопка сектора

  • Готово — сектор можно развернуть
  • Наведение — ждёт Ctrl+Click
  • Активна: Ns — сектор уже работает
  • CD: Ns — сектор перезаряжается

Support abilities у боевых пакетов

  • Нет сектора — сектор нужен, но не выставлен
  • Личный КД: Ns — cooldown именно этой кнопки
  • Общий КД: Ns — общий cooldown пакета поддержки

Важно:

  • таймер сектора живёт только на кнопке сектора;
  • если у ability одновременно активны локальный и общий cooldown, на кнопке показывается больший из двух.

Support abilities у Logistics

У логистики нет статусов сектора.

Ты видишь только:

  • больший из Личный КД: Ns и Общий КД: Ns, если активны оба
  • Личный КД: Ns, если активен только локальный cooldown
  • Общий КД: Ns, если активен только общий cooldown
  • Готово
