# RTO Support: памятка игроку

## 1. Что у тебя есть

RTO получает:

- `RTO binoculars`
- `RTO sling pouch` с жесткой привязкой к этим биноклям

Кнопки RTO видны только пока бинокль у тебя в руке.

## 2. Как выбирать пакеты поддержки

Теперь RTO может держать **до двух уникальных пакетов одновременно**.

Правила выбора:

1. возьми `RTO binoculars` в руку;
2. нажми `Выбрать пакеты поддержки`;
3. выбери один или два пакета;
4. полный сброс обоих слотов станет доступен через `60 минут` от первого выбора в текущем цикле.

Важно:

- дубликаты пакетов выбрать нельзя;
- второй слот необязателен;
- если выбран только один боевой пакет с сектором, его `cooldown сектора` уменьшается в `2 раза`;
- если потом выбрать второй пакет, solo-бонус пропадает и возвращается обычный cooldown сектора.

## 4. Боевые пакеты через сектор

Для `Mortar`, `CAS` и `Heavy Strike` цикл такой:

1. возьми бинокль в руку;
2. выбери пакет;
3. нажми кнопку сектора этого пакета;
4. во время зума через бинокль поставь сектор `Ctrl+Click`;
5. нажми нужную ability;
6. снова наведи точку через бинокль `Ctrl+Click`.

Коротко:

`взял бинокль -> выбрал пакет -> развернул сектор -> выбрал ability -> навёл через бинокль`

Важно:

- одновременно может существовать только **один** активный сектор;
- sector-based abilities работают только через сектор **своего** пакета;
- сектор `Mortar` не подходит для `CAS`, и наоборот;
- после окончания сектора запускается общий zone cooldown и личный zone cooldown пакета-источника.

## 6. Как работают cooldown'ы

У RTO теперь есть три разных вида cooldown'ов:

### `Общий КД пакета`

- общий для всех support-ability внутри одного пакета;
- вызов из пакета A не блокирует пакет B.

### `Личный КД ability`

- относится только к одной конкретной кнопке.

### `КД сектора`

- есть только у боевых пакетов;
- состоит из общего zone cooldown и личного zone cooldown пакета-источника;
- если выбран только один боевой пакет, effective cooldown сектора уменьшается в `2 раза`.

## 7. Utility-режимы

### `Координаты`

- это не ability с cooldown;
- это постоянный режим;
- `Ctrl+Click` просто пишет координаты точки;
- лазерной точки у этого режима нет.

### `Лазерная отметка`

- это тоже постоянный режим без cooldown;
- `Ctrl+Click` ставит или переносит живую подсветку;
- лазер держится только пока ты реально ведешь цель через бинокль.

## 8. Как читать кнопки

### Кнопка сектора

- `Готово` - сектор можно развернуть
- `Наведение` - ждёт `Ctrl+Click`
- `Активна: Ns` - сектор уже работает
- `Общий КД: Ns` - идет общий zone cooldown
- `Личный КД: Ns` - идет личный zone cooldown этого пакета

### Support abilities у боевых пакетов

- `Нет сектора` - сектор нужен, но не выставлен
- `Личный КД: Ns` - cooldown именно этой кнопки
- `Общий КД: Ns` - общий cooldown пакета поддержки

Если одновременно активны личный и общий cooldown ability, на кнопке показывается больший из двух.

### Support abilities у пакетов без сектора

У `Logistics`, `Medical`, `Technical` и HALO utility-пакетов нет статусов сектора.

Ты видишь только:

- больший из `Личный КД: Ns` и `Общий КД: Ns`, если активны оба
- `Личный КД: Ns`, если активен только личный cooldown
- `Общий КД: Ns`, если активен только общий cooldown пакета
- `Готово`

## 9. Что показывает examine у бинокля

Examine у `RTO binoculars` объясняет:

- какие пакеты сейчас выбраны;
- кто сейчас владеет активным сектором;
- сколько осталось до полного reset слотов;
- активен ли solo-бонус на cooldown сектора;
- какой режим сейчас armed;
- активна ли ручная лазерная отметка.
